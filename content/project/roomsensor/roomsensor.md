# Overview

Inside our new home we have an KNX <https://www.knx.org> automation bus. I have not found an nice looking "RoomSensor" which has the following sensor equipment:

* temperature
* humidity
* brightness
* passive infrared motion (PIR)
* smoke detector
* CO2
